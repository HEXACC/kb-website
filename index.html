    课表查询

    「课表」使用了学校公开的课程数据，进行整理并展示。

    支持今日课表和全部课表两种展示方式，并且，你还可以将当日课表分享给其他同学。

    想使用课程数据？

    Api Center开放了烟台大学及烟台大学文经学院所有院系的课程数据，你可以访问这里来查询如何获取这些数据。

    了解分享课表的更多功能»

    成绩查询

    「课表」支持期末成绩查询，第一次查询时需要绑定学号。(1)

    我的密码保存在哪里了？

    密码只会保存在你的本地存储上，不会再任何服务器进行保存。

    成绩的数据来源可靠吗？

    成绩数据全部抓取自学校的官方成绩查询公众号，所以可以很放心数据来源。

  • (1) 目前学校重写了第三方成绩查询接口，我们暂时关闭了成绩查询相关功能。

    • 了解成绩查询的更多功能»

    空闲教室

    「课表」不仅支持成绩和课程查询，还支持空闲教室查询，方便你找到自习教室。

    想使用空闲教室数据？

    Api Center开放了烟台大学空闲教室数据，你可以访问这里来查询如何获取这些数据。
