    「课表」V3.0

    提供烟台大学在校生班级课表、空闲教室查询及成绩查询服务。

    扫码体验

    今日

    「今日」视图由不同的卡片构成，你可以在不同的卡片中查看今日所有的信息。(1)

    0 通知卡片

    「课表」的通知卡片，在特定节日中会显示，你也可以选择永久关闭。

    1 日历卡片

    在日历卡片中，同学可以很方便的了解到今天是周几，是第几周。同时，你可以点击不同的日历查看当前日期下的课程信息。

    2 功能卡片

    所有的功能将在功能卡片中呈现，同学可以选择不同的功能。

    3 课程卡片

    当你在「我的」页面中绑定了我的班级信息后，今天的课程将在这里呈现出来。

    如果你不想使用本班级的课表信息，可以在「我的」页面手动设置班级信息。（注意，若手动设置后想要恢复自动设置需要清除数据后重启小程序生效）

  • (1) v2.0.4版本起，你可以主动关闭所有你想隐藏的卡片。

    • 课表查询

    「课表」使用了学校公开的课程数据，进行整理并展示。

    支持今日课表和全部课表两种展示方式，并且，你还可以将当日课表分享给其他同学。

    想使用课程数据？

    Api Center开放了烟台大学及烟台大学文经学院所有院系的课程数据，你可以访问这里来查询如何获取这些数据。

    了解分享课表的更多功能»

    教务系统

    「课表」3.0中增加了教务系统，支持期末成绩查询、选课信息查询、考试安排等。(1)

    我的密码保存在哪里了？

    密码只会保存在你的本地存储上，不会再任何服务器进行保存。当你选择记住密码记时默认同意《课表隐私协议》。

    成绩的数据来源可靠吗？

    成绩数据全部来源学校官方教务系统，所以可以很放心数据来源。

  • (1) 建议第一次使用「课表」时进行登录绑定学号。

    • 了解成绩查询的更多功能»

    空闲教室

    「课表」支持空闲教室查询，可查询，方便你找到自习教室。

    想使用空闲教室数据？

    Api Center开放了烟台大学空闲教室数据，你可以访问这里来查询如何获取这些数据。

    了解空闲教室更多信息»

    校历

    我们将烟台大学2017-2018第一学期6个月的校历放在了「课表」中。

    学期中假期和节日都将在日期中显示。

    同时，点击右上角的分享你可以很方便的将校历分享给其他同学。

    「我的」视图

    在「我的」视图中，可以进行一些相关设置或者清除本地存储

    v2.1.0开始，「课表」第一次使用时将不再自动获取用户信息，需要用户进行手动授权。

    分享

    在「课表」中，你可以几乎将所有的页面分享出去。

    得益于微信小程序的参数页面，同学问你下节课在哪上课，你可以直接把当天的课表发给他，朋友问你想去哪里去上自习，直接发给她空闲教室，让她去选择喜欢的教室。

    目前支持「课表」分享的页面有：某个班级的课表、校历、某节课空闲教室。

    了解分享的更多功能»

    新生查询(1)

    新学期新气象！2017级新生可使用「课表」查询宿舍、班级信息。

  • (1) 临时性功能，开学季之后将取消此功能。

    • 烟雨楼台教务 Fork Version

    烟雨楼台教务是为烟雨楼台公众号而制作的「课表」小程序的分支版本。

    它和「课表」的主体功能一致，是烟雨楼台公众号的定制版本。
